# DevOps home lab

System Requirements:

- 8 CPU
- 12 Gi of memory
- 60 Gi of disks

For arranging home lab, just run scripts one after another, keeping in mind its prefix (`01_`, `02_` etc.)

Components of home lab:
- MetalLB
- Traefik ingress controller
- ArgoCD
- Gitlab
- VictoriaMetrics + Grafana
...
